# Story X — 라이브 실증 공개 패키지

> Story X 는 한국어 장편 연재 스튜디오다. **연속성(continuity)을 편집 마무리가 아니라 제품 요건으로 다룬다** — 설정·인물·약속이 회차를 거치며 무너지지 않도록 시스템이 기억하고, 검토하고, 충돌을 드러낸다.

이 폴더는 2026년 6월에 수행한 **라이브 실증 테스트의 기록을 선별 공개**한 것이다. 마케팅 문구가 아니라, 실제로 작품을 만들면서 시스템이 무엇을 잡았고 무엇에 실패했는지를 그대로 보여주는 로그다.

## 무엇을 실증했나

- **로맨스판타지 장편 1편을 23화 완권**으로 완성했다 — 1화부터 23화까지 확정 설정(캐논)의 드리프트 0.
- 회차마다 **5인 AI 검토진**(쇼러너·캐릭터·세계관·장르·연속성)이 실제 본문을 읽고 판정했다. 검토진은 캐논 충돌을 발견하면 **"출고 불가"** 까지 선언한다.
- 소설 외에 **만화·에세이·오디오·학술 4개 매체**에서도 같은 코어(캐논·인물 추출·검토)가 작동함을 확인했다.
- AI 연재의 고질병 — **"이야기가 틀리지는 않는데 끝나지를 않는다"** — 를 정면으로 발견했고, 작가 결정(갈림길)과 진도 인터뷰로 끊는 메커니즘까지 실증했다.

## 정직성에 대하여

이 기록은 **제작자가 직접 수행한 내부 실증**이다. 외부 작가의 독립 검증이 아니다. 테스트 페르소나(웹소설 다독자, 헌터물 독자 등)를 설정하고 실제 UI 를 직접 조작해 작품을 만들었으며, 잘 된 것과 실패한 것을 모두 기록했다. 실패 사례 — 의도 메모 오염, 인물 이름 드리프트, 캐논 충돌 사고 — 는 본문에 그대로 남겨 두었다. 그 실패들이 시스템이 왜 이렇게 생겼는지를 설명하기 때문이다.

다음 단계는 외부 작가의 독립 완주다.

## 베타 작가 모집

연재 경험이 있는 작가 3~5명을 클로즈드 베타로 모집한다.

- **무엇을 하나** — Story X 로 작품 1편(10화 이상)을 직접 완성한다. 매체는 소설·에세이·만화·오디오 모두 가능.
- **비용** — LLM 비용은 본인 키(BYOK) 사용. 도구 자체는 무료.
