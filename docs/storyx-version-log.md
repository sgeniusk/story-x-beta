## 현재 버전

### Alpha v0.9.0 · Closed Craft Loop

- 날짜: 2026-05-17
- 검증: `npm test` 28 files / 130 tests, `npm run build` pass

이번 마일스톤에서 생성·검토·인터뷰가 모두 **작가가 쓴 내용을 기준으로 한 루프**로 돌게 되었고, 대중성·작품성 craft 원리가 그 루프 안에서 작동하게 되었다.

- **닫힌 창작 루프**: 검토에서 승인한 기억 후보가 실제 작품 캐논으로 반영된다. 생성 → 검토 → 승인 → 캐논 → 다음 생성이 한 순환으로 이어진다.
- **동적 작가 인터뷰**: 작품 인터뷰가 매체별 고정 질문 대신, 작가가 쓴 자유 서술을 읽고 그 작품에 맞는 질문을 만든다. 인터뷰어는 유명 작가 오마주 페르소나(아가타 크리스·도스토옙·무라카메·버지니아 울브·유발 하리 등)로 묻는다.
- **craft 원리 적용**: 대중성·작품성 리서치를 반영해 Story Contract를 표면 약속·심층 질문·무게중심으로 확장하고, 검토 기준과 문체 평가에 craft 점검을 더했다.
- **장편·영속성**: 장편 10화 회귀 하네스, 컨텍스트 예산 제한, 작품 버전 히스토리·복원, 직접 편집 diff, 매체별(소설/에세이) 생성·검토 분기를 더했다.

다음: 외부 사용자를 위한 서버측 LLM 운영(2단계 E)을 준비한다.
