## 현재 버전

### Alpha v0.10.0 · Quiet Studio

- 날짜: 2026-05-18
- 검증: `npm test` 28 files / 127 tests, `npm run build` pass

이번 마일스톤에서 제품의 모든 화면을 **Notion 디자인 시스템**으로 다시 세우고, 에디터 작업 구조를 작가가 원고에 집중하도록 정리했다.

- **전 화면 리디자인**: 랜딩·프로젝트 목록·홈 플로우·에디터·로그인을 라이트 캔버스·purple primary·사각 버튼의 Notion 톤으로 통일했다.
- **에디터 구조 개편**: 원고가 화면의 주인공이 되도록 얇은 툴스트립과 상단바 액션 축소, 트랙 전환 페이드를 적용했다. 좌측 레일은 단일 스크롤·카드형 회차 목록·캐논 건강도 바, 바이블 목차는 갤러리 카드 그리드로 바꿨다.
- **작가진 다이얼로그**: 헤더·본문 스크롤·입력 3분할로 나눠 조언이 길어도 입력창이 항상 보인다.
- **인터뷰·브랜드**: 작가 인터뷰 질문을 매체별 6~10개로 늘리고 자유 메모를 마지막 단계로 분리했다. 브랜드 심볼 로고를 모든 화면 상단에 복원했다.

다음: 만화 제작실과 외부 사용자를 위한 서버측 LLM 운영(2단계 E)을 준비한다.
