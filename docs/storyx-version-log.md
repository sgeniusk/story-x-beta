## 현재 버전

### Alpha v0.8.0 · Real LLM Writing Loop

- 날짜: 2026-05-16
- 검증: `npm test` 24 files / 113 tests, `npm run build` pass

이번 마일스톤에서 초안 생성과 작가진 검토가 mock에서 **실제 Claude 구독 호출**로 바뀌었다. 토큰당 API 과금이 아니라 사용자의 Claude 구독(`claude auth login`) 한도 안에서 동작한다.

- **초안 생성**: 편집기의 "초안 생성"이 로컬 dev 브리지(`/api/draft`)를 거쳐 실제 LLM 회차를 만든다. 배포본에서는 기본 생성으로 자동 폴백한다.
- **작가진 검토**: "흐름 검증 / 결 점검"이 쇼러너·캐릭터·배경·장르·연속성 5개 에이전트의 실제 판정을 받는다(`/api/review`).
- **연속성 컨텍스트**: 2화 이상을 생성할 때 확정 캐논·인물·세계 규칙·열린 떡밥을 LLM에 함께 넘겨 회차 간 연속성을 지킨다.
- **버그 수정**: `--max-budget-usd` 예산 상한이 긴 생성을 중간에 죽이던 문제 제거(구독은 토큰 과금 없음), 편집/바이블 트랙 전환 시 가로 흔들림 제거.

다음: 검토에서 승인한 기억 후보가 실제 작품 캐논·바이블로 반영되는 루프를 닫는다(로드맵 참고).
