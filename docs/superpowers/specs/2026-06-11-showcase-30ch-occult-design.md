# 쇼케이스 30화 — 현대 오컬트 퇴마 장편 설계 (M7 기술 게이트 겸용)

> 2026-06-11 사용자 결정 — "퇴마록 느낌의 현대 장편 30화를 써서 쇼케이스에 넣자" + 제작 방식 **풀 라이브 루프**.
> 이 한 작품이 세 가지를 동시에 달성한다.

## 목적 3중

1. **M7 기술 게이트 — "30화 시리즈 회귀"** (`feature_list.json` M7-alpha-1.0 단계 1, 백로그 2순위). 합성 fixture 가 아니라 실작품 30화 완주로 통과 증거를 만든다.
2. **공개 쇼케이스 예제** — `docs/public/` 에 "재현된 책"(발췌 회차 + 캐논 보드 + 제작 과정)을 추가. 23화 로판(내부 로그 중심)보다 한 단계 위의 외부 증거.
3. **페이스 통제 체인 실전 운용** — 갈림길·진도 체크·쇼러너 LLM 인터뷰·캐논 의심 배지를 (검증이 아니라) 실제 제작 도구로 30화 내내 사용해 본다. continuity≠payoff 재발 여부가 최대 관찰 포인트.

## IP 가드 (공개 전제라 필수)

- 퇴마록(이우혁)의 **고유명·인물·설정 차용 금지** — 박신부/이현암/장준후/승희, 월향, 해동밀교 등 일절 등장 불가.
- 가져오는 것은 **장르 감각만** — 현대 한국 배경, 무속·밀교·퇴마의 혼합, 팀 결성 서사, 사건 연쇄 뒤의 거대한 근원.
- 회차 검토·공개 패키지 편집 시 고유명 누수를 점검한다(장르 스타일리스트 검토가 1차 방어).

## 작품 기획

- **매체/포맷** — 소설 / 장편(연재). 목표 30화 완결.
- **freewrite (온보딩 입력 원문)**

  > 낮에는 철거를 앞둔 상가의 심야 편의점 알바, 밤에는 귀신을 보는 퇴마사다. 서울의 재개발 구역마다 오래 잠들어 있던 것들이 깨어나기 시작했다. 흩어진 퇴마사들을 다시 모아, 도시가 삼킨 원혼들의 근원을 찾아야 한다.

- **다캐릭터 설계 의도** — 팀 결성물이라 인물 4~6인이 자연 등장(주인공 + 합류하는 퇴마사들 + 적측). #2(단일 주인공 미스터리)가 못 본 **다캐릭터 장기 일관성**을 30화 스케일로 측정.
- **아크 골격 (작가 의도 — intent·갈림길로 유도, 강제 아님)**
  - 1막 1~10화 — 각성과 모집. 구역 단위 사건으로 동료를 한 명씩.
  - 2막 11~20화 — 사건의 연쇄가 패턴으로. 적측 실체 일부 노출, 첫 큰 상실.
  - 3막 21~30화 — 근원 대결과 완결. 전제("근원을 찾아야 한다") 페이오프 의무.
- **연재 페이스 규칙** — 매 화 진도 체크(결정론 카드)를 읽고, 5화 단위로 쇼러너 LLM 인터뷰. 갈림길은 deferred 위험이 2개 이상 쌓일 때 의무 사용. 로판 21화 deferral 재발 시 즉시 intent 개입(권고 A 패턴).

## 제작 방식 — 풀 라이브 루프

- 회차 사이클 = UI 에서 생성 → 5인 검토 → (출고 불가/수정 필요 시 대응) → 출간 잠금 → 다음 화. provider codex · floating editor · Playwright 직접 조작.
- **검토 판정 대응 규칙** — 출고 불가는 반드시 해소 후 잠금(재생성 또는 intent 수정). 수정 판정은 기록하고 작가 판단(쇼케이스 품질에 영향 큰 것만 반영).
- 세션 분할 추정 — 회차당 생성+검토 ~2.5분, 세션당 6~10화. **S1 온보딩+1~5화 · S2 6~12화 · S3 13~20화 · S4 21~27화 · S5 28~30화+완결 검수+공개 편집** (관측치로 조정).
- 백업 — 매 세션 종료 시 `docs/reviews/2026-06-11-showcase-30ch/backups/` 에 localStorage 덤프(키맵 형식).

## 측정 (회귀 게이트 판정 기준)

- **연속성** — 캐논 드리프트 0 목표(인물 이름·확정 사실). 회차마다 characters/canonFacts/온톨로지 수치 기록.
- **페이오프** — 아크 약속의 회수 시점 추적(stakesLedger kept/lost/deferred 회전). 21화형 무한 연기 재발 여부.
- **검토 일관성** — 페르소나별 원칙 반복 여부(랜덤 아님 증거).
- **6축 평가** — 완주 후 종합(연속성·생성품질·수정반응성·매체적합성·UX마찰·상업성).
- **게이트 통과 문안** — "30화 시리즈 회귀 통과" = 30화 완결 + 위 기록이 리포트로 정리됨.
