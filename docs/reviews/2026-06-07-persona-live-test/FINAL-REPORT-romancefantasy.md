## 차별점 입증 — 무엇이 증명됐나
1. **캐논화 후 고정** — 인물이 canonFacts 에 들어가면(캐논화) digest 를 통해 다음 회차가 고정. 루시안 7연속, 백작부인 reveal 정합. P5 이전 "캐논화 *이전*" 인물만 드리프트.
2. **검토 일관성(반복 가능성)** — 페르소나가 자기 원칙을 캐논·세계규칙 근거로 회차마다 반복: 세계 키퍼 "비용/메커니즘" **ch9~17 9연속**(ch19 본문 자가수용으로 종결), 연속성 "확정 강도" 다회, 캐릭터 "신뢰 속도" 다회. 랜덤 아님 = 차별점 신뢰성의 핵심.
3. **캐논이 추리 엔진** — 고정 이름(레나=W·루시안=L)이 ch11 첫배신자 소거 논리로, ch12 페이오프(죽은 벨로트 레오르)로 작동. 연속성이 "보존"을 넘어 "전개 제약".
4. **품질 게이트 본문반응** — hook 게이트가 회차별 실제 본문 따라 5/8↔4/8↔6/8 변동(rank3 "게이트가 본문을 읽는다" 라이브). ch12 게이트(엄격)↔페르소나(통과) 분기 = 결정론·LLM 두 렌즈 상보성.
5. **미스터리 자가교정** — 검토 우려를 후속 회차 본문이 캐논으로 해소(여러 사례).
6. **P1(쇼러너 빈 응답) 0/13** — 이번 세션 무재현(간헐성 잔존하나 추세 양호).

## ★ 최대 발견 — continuity ≠ payoff
21화 내내 **연속성·회차 모멘텀은 완벽하나 전제("가문을 살리고 운명을 바꾼다"·3년 뒤 처형 회피)의 페이오프가 0 진전**. codex 가 배신자 정체를 ch19·20·21 한 겹씩 무한 연기(미스터리 루프 자가증식). 리아나는 21화 동안 *수사만* 하고 운명을 바꾸는 *행동*은 안 함.
- **시스템 갭** — 회차 게이트(hook_first/last)는 있으나 **작품(아크) 게이트 = audiencePromise·dramaticQuestion 의 실제 진척을 강제하는 장치가 없다.** 쇼러너는 약하게만 신호("압력이 추론으로만 진행")하고 통과시킴.
- **의미** — Story X 는 "이야기를 *틀리지 않게*" 만드는 데는 탁월하나, "이야기를 *끝나게/보상하게*" 만드는 강제는 약하다. 1.0 시장증명(rank6)의 핵심 과제.

## 개선 — 완료 + 검증
- **P6** `extractEntityName` 명명 계사 "(이)라는" → "레오르 벨로트라"→"레오르 벨로트". **P5** `extractCharacterNames` 서술부 명명 추출 → 루시안 등 승격. **relations** `extractRelation`+`linkRelationsFromCanon` → "A의 [관계] 이름은 B" 엣지. TDD(RED→GREEN), 325 tests, 커밋 `e4a2ea2`.
- **라이브 검증** — ch19 character 캐논 등장 시 "백작부인" 클린 승격(조사버그 0·canonAnchors 보존) 실증.

## 권고 (우선순위)
1. **[높음] 아크 페이오프 게이트** — audiencePromise/dramaticQuestion 의 진척을 회차마다 측정·강제. 예: N회차 이상 핵심 질문이 "발견"에만 머물고 "행동/대가/전환"이 없으면 경고. (continuity≠payoff 갭의 직접 처방. rank6 시장증명과 직결.)
2. **[중] codex 과확정 억제** — 생성 프롬프트에 "미해결 미스터리는 추론 톤 유지(확정 금지)" 지시. ch15 5인 수렴 포착·ch19 자가회피 사례로 보아 효과 기대. (고위험 — before/after 품질 비교 필요.)
3. **[중] 캐릭터 캐논 밀도** — codex 가 `owner:'character'` 캐논을 불규칙 산출(ch14~18 5연속 0개) → 관계 그래프 희소. 생성 프롬프트에 "새 인물 등장 시 character 캐논 명시" 유도. (plot 캐논 스캔 자동승격은 오탐 위험으로 비권장.)
4. **[중] hook 게이트 생성 반영** — hook_first_300/last_200 빈번 실패 → 생성 시 후크 강화 지시.
5. **[낮음] P1 빈응답 가드**(0/13 무재현) · **floating 2c·2d**(데이터/출간) · MCP 스크린샷 대체(evaluate 정본).
