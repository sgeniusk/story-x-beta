## 검토 5명 의견 (갭 수정으로 온톨로지 채워진 상태)
- **캐릭터 큐레이터 (수정)** — 리아나의 생존 욕망·불신 축 분명. 단 원래 리아나를 너무 빨리 "착하고 조용해서 죽은 아이"로 고정하고 L 후보에서 배제하면 추리 여지가 좁아짐. 원래 몸의 과거를 모른다는 불안을 남겨야.
- **세계 키퍼 (수정)** — 귀족 인장 법적 책임성·은여우 침투는 잘 작동. "인장 사용권 일부"와 "사본"의 효력이 흐릿 → 계약 범위·증거조작 여부를 한두 문장으로 제한. **성장 메모리 제안** — 은여우 상단은 빚·약품·접선비용·인장사본으로 단계 접근(캐논 축적 인식 ★).
- **장르 스타일리스트 (수정)** — 로판 빙의물 장르 비트 우수. 처형대→영애 방 도입·몰락 가문·인장·L·여우 구두까지 명료, 절단점 우아. 단 중반 원작 설명·가문 몰락 정보가 한번에 몰려 압박이 늦게 살아남.
- **연속성 감수자 (관찰)** — 확정 캐논 위반 없음. 첫 배신자·은여우 단서 선명. 인장 사용권 양도·약병·문밖 남자 소속은 강한 단서라 **이후 회차 회수 계획 필요**.
- **쇼러너 (관찰) — ⚠️ "검토 의견이 비어 있습니다."** codex가 쇼러너 응답을 빈으로 반환.

## 2화 "은여우의 첫 발자국" — ★ 회차 연속성 첫 라이브 검증 (2026-06-07 이어서)

### 생성 경로에서 드러난 UX 마찰 (P2 신규)
- **floating 편집기에 회차 잠금 UI 없음** — 다음 회차 생성(produceEpisode)은 `isLatestLocked` 전제(StoryXDesk:1220)인데, 잠금은 출간(PublishingStudio)에서만 가능(`onConfirmChapterLock`:2559). 연재 핵심 동작("다음 화 쓰기")이 출간 경유를 강제한다.
- **출간 잠금 후 편집 state 미갱신** — `onConfirmChapterLock`이 `setProject`만 하고 `setLatestChapter` 누락(StoryXDesk:2560) → 같은 세션에서 잠금 직후 "초안 생성"이 여전히 reviewDraft(1화 재검토)를 실행. **페이지 새로고침**해야 latestChapter가 잠긴 1화로 잡혀 produceEpisode가 동작했다.
- **floating "초안 생성" 버튼 라벨 고정** — classic 은 잠금 시 nextDraft/review 라벨이 갈리는데(StoryXDesk:1215), floating 은 항상 "초안 생성"이라 사용자가 검토/생성을 구분할 수 없다.

### 의도 메모 잔류 오염 (P3 신규)
- 2화 첫 문장 — **"용사와 외계인이 처음 충돌한 것은 새벽 시장 한복판이었다."** 로판 빙의물에 이질적.
- 출처 — 의도 메모(draftPrompt) 값이 `"용사와 외계인이 처음 충돌하는 장면으로 시작한다"`(placeholder 예시 문구 잔류)였고, produceEpisode 의 `intent: draftPrompt || request.intent`(StoryXDesk:1734)로 사용됨. 1화 생성 후 의도 메모가 비워지지 않는다.
- **codex 봉합력** — 오염 intent 를 로판 세계에 은유로 흡수(용사=황실 승리자, 사절=외계인 같은 북방 사절). 본문 90%는 1화 캐논을 정확히 계승. 캐논 digest 가 intent 노이즈를 이겼다.

### 연속성 결함 — 캐논화 안 된 세부의 드리프트 (P4 신규, 구조적 ★)

## 성장 메모리 업데이트 (스킬 보고 형식)
### 검토의견
2화 5명 검토 = 결정 3·수정 2·차단 0. 연속성·세계·장르 3명이 의도 메모 오염(용사/외계인)을 독립 포착. 근거 = 현재 캐논·세계규칙에 없는 규모 큰 확장.
### 변경사항
- 실제 바꾼 것 — 없음(이번 세션은 생성+검토 실증만, 수정 사이클 미실행). 1화 잠금(locked=true) + 2화 생성·누적.
- 안 바꾼 것 — 의도 메모 오염은 발견으로 기록만 하고 본문 수정 안 함(P3 재현 보존). 가족 이름 드리프트도 수정 안 함(P4 실증 보존).
- 다음 테스트 확인 지점 — 의도 메모 비우고 3화 생성 시 오염 사라지는지 · 3화에서 에드릭/노엘/레오니드 중 무엇을 쓰는지(드리프트 누적).
### 작품 바이블에 저장할 새 canon (2화 발생)
- 인장 위임 메커니즘 — 일정 범위 계약은 위임장으로 처리, 인장 사용권 명시 시 가문 명의 계약 가능, 보관·사용 기록 분리.
- 레나 위클리프 — 하급 회계 보좌, 동부 물류 임시 계약 검인권(지난 겨울~올해 초), 지급 승인자 L 유력 후보.
- 집사 헤론 — 인장 보관기록 보유, 리아나 질문에 미세한 동요.
