# 예비비행 — 6-페르소나 로컬 Codex 폭 검증 (context-notes)

작성 2026-07-08. 외부 작가 실증(M7 단계2) 착수 전, 사용자가 직접 설계한 6개 조건을 로컬 Codex로 돌려 **제품 견고성·매체 폭**을 훑는 예비비행.

## 목적과 경계 (중요)

- 이것은 **제품 견고성/폭 검증**이다 — 여러 장르·매체·조건에서 완성 루프가 안 깨지나, 게이트가 실제로 작동하나.
- 이것은 **M7 단계2(외부 시장 신호)를 대체하지 않는다**. 단계2는 제작자 개입 없는 외부 작가를 요구한다(docs/decisions/2026-06-10-market-proof-1.0.md). 여기 페르소나 조건은 사용자가 설계한 가설이라 "낯선 사람이 돈 낼까"는 못 잰다.
- 자리매김 = 외부 작가 붙이기 전 예비비행. 여기서 나온 버그·이탈지점을 고친 뒤 진짜 사람을 부른다.

## 실행 방식 (사용자 결정)

- **CLI 배치** — `node tools/storyx.mjs draft --provider codex` 로 6개를 스크립트 순차 생성. UI 여정은 first-run E2E(2026-07-08, 반환실의 초승달)로 이미 1회 증명됨.
- BYOK 불필요 — 로컬 Codex(M12 배선)라 키 입력 없이 무과금.
- 각 초안 후 `storyx-persona-review` 스킬로 페르소나 검토 → findings 남김.
