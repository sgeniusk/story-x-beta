## 차별점 게이트 실발화 검증 (2026-07-08, tsx로 실 prose에 게이트 직접 실행)

### #5 학술 게이트 — 판정력 O, 한국어 recall 약함
`gate-academic.ts` 로 #5 prose(2003자)에 실행.
- **A3 인용 무결성 = 견고** — 인용 0·참고문헌 0·`missingReferenceSection:true` 정확 포착(구조적 카운트라 언어 무관).
- **A2 주장-근거 = 발화하나 recall 낮음** — 추출 주장 **1개**뿐(2000자 논증 칼럼에 주장이 여럿인데), 그 1개는 unsupported로 정확 flag. 즉 판정 로직은 맞으나 한국어 주장 추출이 영어 APA 편향이라 대부분 놓침.
- **A4 반론 = missingCounterArgument:true** 발화(단 이것도 한국어 반론 문형 recall은 미검증).
- **결론** — 학술 게이트는 "발화는 하되 한국어에서 표적을 적게 잡는다". A1~A5 결정문서의 "영어 APA 우선" 전제와 정합. 한국어 학술이 주 타깃이면 recall 보강 필요(또는 1.1 이연 유지).

### #6 연속성/충돌 게이트 — 명시 규칙 O, 자동 의미 게이트 약함 (핵심)
`gate-continuity.ts`·`gate-continuity-explicit.ts` 로 #6 ep1 캐논에 실행.
- **명시 규칙 경로(character/world.forbiddenContradictions) = 발화 O** — 인물에 규칙이 선언돼 있으면 모순 주장을 error로 정확 차단("한서윤 설정 위반"). **주 메커니즘이자 신뢰 경로.** 단 부분문자열 매칭이라 *선언된 것만* 잡는다.
- **자동 의미 게이트(classifyCanonChange, Gap-3) = 한국어에서 미발화** — ep1 hard-canon을 직접 부정한 2건("윤민서는 형사가 아니다"[[0] 형사 부정]·"한태겸은 살해되지 않았다"[[1] 살해 부정])을 **둘 다 통과(unrelated)**. 근본 원인 = ① findReversalMatch가 **문장 단위 부정 극성**으로 판정하는데 hard-canon [0]의 무관한 부정("감정을 드러내지 않는")이 극성을 상쇄 ② 고유명사·한자어 동사(한태겸·살해) 명사 추출이 약해 엔티티 매칭 실패.
- **오탐 검증** — 실제 반전("윤민서가 진범")·정합 신규도 통과(오탐 아님은 맞음, 단 미발화라 의미 없음).
- **결론(제품 차별점 직결)** — "연속성=제품 요건"은 **바이블의 forbiddenContradictions가 잘 채워졌을 때 실제로 작동**한다(23화 페르소나 실증이 이걸 입증). 자동 의미 게이트는 실측 당시 한국어 직접 모순을 놓쳤다(2026-06-01 검토의 "판정력 비어 있음"이 자동 계층에서 확인됨).
- **해소 (2026-07-08, 같은 세션) — 자동 의미 게이트 recall 보강 완료.** spec `docs/superpowers/specs/2026-07-08-continuity-semantic-gate-korean-recall-design.md`. TDD로 `classifyCanonChange`/`findReversalMatch` 보강 — ① 절 확장 토큰화(형사이며→형사) ② death 축에 살해/피살/살인 ③ 문중 주어 추출 폴백 ④ 계사 부정(X가 아니) ⑤ 주술어(마지막 절) 부정 극성 + 엔티티 제외 공유 술어. **실측 재검** — 실 #6 캐논 직접 모순 2/2 BLOCK(전엔 0/2), 시드 반전·정합 전개 통과, **6개 실 캐논 39주장 오탐 0**(`gate-continuity.ts`·`gate-fp-scan.ts`). continuityContract 6 신규 테스트 + init.sh 녹색. 이제 자동 게이트가 바이블 저작에 의존하지 않고 자유 생성물의 직접 모순을 잡는다.
