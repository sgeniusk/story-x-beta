# STORYSCORE_V0_1 A/B 리포트 — 실험군(Story X+codex) vs 통제군(맨 Claude)

> 2026-06-11. 같은 기획 입력(freewrite+인터뷰 8답+아크 의도)으로 만든 30화 완결 장편 2편을 같은 루브릭으로 채점했다.
> 작품 A = 통제군 『잿우물』(맨 Claude 단일 컨텍스트 집필, `control-claude/`) · 작품 B = 실험군 "철거 전야의 이름"(Story X 풀 라이브, `backups/occult-ch30-complete.json`).
> 심사자 한계 — 심사자(Claude)가 통제군의 저자이자 실험군의 운영자라 완전 맹검이 불가능했다. 라벨 중립화·동일 루브릭·근거 인용 의무로 보정했으나 외부 재채점을 권장한다.

**한 줄 결론 — 이번 조건(30화·단일 세션 스케일)에서는 잘 설계된 맨 Claude가 Story X+codex 라이브 연재를 이겼다.** 단 조건의 비대칭(아래 교란 변수)이 커서 "하네스가 무가치하다"가 아니라 "하네스의 다음 과제가 무엇인지"를 가리키는 결과로 읽어야 한다.

## 축별 근거 (요약 — 인용은 표본 기준)

### 연속성 — A 5점 / B 4점
- **A 5점.** 후반 표본이 초반 설정을 정밀 회수한다. 1화 "근무표를 뒤져 점주 이름을 확인하고 손등에 볼펜으로 적었다. 곽필성." → 30화 "손등에 이름을 적는 버릇은 없어졌다. 대신 계산대 아래 수첩이 한 권". 13화 민규원의 "거짓말이라도 6년 만에 들은 딸 목소리였습니다" → 23화 가짜 딸 판별 근거("우리 채아는 그날 목이 쉬어서 노래를 못 불렀어")가 인물 고유 기억으로 작동. 표본·말미 시퀀스에서 모순 미발견.
- **B 4점.** 서사 내 모순은 0 — 특히 16화 대가("네, 라는 한 음절이 목 안에서 사라졌다") → 23화("고개를 들지 못했다. …첫 반응은 이미 잃어버렸으니까") → 32화("잠깐 늦게 돌아섰다. …그러나 그는 정확히 돌아섰다")로 **대가의 누적이 회차를 가로질러 정밀하게 일관**된 것은 최상급. 감점 사유는 **회차 번호 표기 드리프트** — 16화 다음이 19화로 점프(17·18 결번), 제목 번호와 실제 위치가 어긋남(독자 가시 흠). 원인은 하네스의 폴백 회차가 episode 번호를 소모한 사고로, 본문 품질 문제가 아니다(부록 참조).

### 페이오프/구조 — A 5점 / B 4점
- **A 5점.** 추출한 약속이 전부 회수되고 회수가 선택·대가를 통과한다. 대표 짝 — 1화 발급 "겨울아."(원혼의 본명 호명) → 29화 회수 "한겨울. 차도현이고요. 둘 다예요." / 13화 민규원의 결핍 → 23화 희생("박자도 못 맞추는 음치의 목소리로, 곰 세 마리를 부르기 시작") — 떡밥(노래)이 인물의 대가로 되돌아오는 모범 회수. 30화가 전제에 정면으로 답한다("온 도시를 뒤져서 찾아낸 게 내 어린 날이었어요").
- **B 4점.** 중심 약속은 회수된다 — 1화 "강이현."(호명) → 32화 "근원은 강이현이 잃어버린 기억이었다"+이름 수용("그러나 그는 정확히 돌아섰다"). 문태겸의 죄(15화 "그 이름은 내가 멈췄다") → 32화 책임 인수. 감점 — 19~27화 구간의 **"0번 소품" 연쇄**(전화함→수건고리→온수밸브→샤워 꼭지→거름망→드라이어)가 발견 축적 패턴으로 회수 속도를 늦췄고(운영자가 페이스 시드·intent로 절단), "담임"의 최종 처리가 표본 범위에서 A의 대응물(백작부인류 명시 결판)만큼 또렷하지 않다.

### 프로즈 품질 — A 4점 / B 4점 (동점)
- **A.** 강점 — 인물 분화(은파의 퉁명, 민규원의 고요, 곽 점주의 생활어)와 장면 구체("잃은 자리는 잃고 나서야 보인다"). 감점 — 30화 에필로그의 해설체("변한 것은 도현이었다. 그는 더 이상 잃는 것을 두려워하지 않았다")가 의미를 직접 말한다.
- **B.** 강점 — 이미지 밀도와 감각 구체("늦춘 시간은 은혜가 아니라 빚이었다", "자기 것이면서도 남의 방에 걸린 외투 같았다"). 감점 — 후반 장면 구조 반복(장소 기억 열림→과거 장면→대가 청구)과 32화의 따옴표 없는 간접 대사 전환(문체 미세 흔들림).

### 연재성/후크 — A 5점 / B 4점 (심사분)
- **A.** 말미 결이 다양하다 — 정보 역전(3화 "그 집 아이는 분명히 죽었다고 들었는데"), 위협 구체화(7화 "방금 넷 됐어요"), 시한(18화 "발파, 모레예요"), 감정 후크(20화 "겨울아. 오지 마"). 다음 회차가 후크를 실제로 이어받는다.
- **B.** 후크 자체는 강하고 누적 설계(첫소리 ㅇ→모음 ㅗ 누적, "회수 예정")가 좋으나, 16~26화 말미가 "새 소품+첫소리" 동일 패턴으로 반복된다.

## 교란 변수 (이 결과를 일반화하기 전에)

1. **모델 비대칭** — 통제군은 Claude(현 최상위급), 실험군 생성은 codex. 이 실험은 "하네스 유무"가 아니라 **"하네스+codex vs 맨 Claude"** 비교다. 하네스 효과를 분리하려면 같은 모델로 양 조건을 돌려야 한다(후속 실험 1순위).
2. **컨텍스트 조건** — 통제군은 30화(58K자)가 단일 컨텍스트에 들어가는 스케일이라 사전 설계·전체 조망이 가능했다. Story X의 설계 목표(컨텍스트가 끊기는 100화+·멀티세션·다작가)는 이번 조건에서 시험되지 않았다. 통제군에 유리한 설계였음을 명시한다.
3. **사전 아크 설계** — 통제군은 집필 전 내부 바이블을 만들고 떡밥 회수를 설계했다(에이전트 자가 보고). 실험군은 회차별 즉흥 생성 + 운영자 페이스 개입이었다. **이 차이가 페이오프 축 점수차의 주범**으로 보인다 — Story X에 "시즌 아크 플래너"(약속·회수 시점의 사전 설계와 강제)가 없다는 갭을 가리킨다.
4. **심사 비맹검** — 상기.

## 실험이 산 것 (Story X 개선 백로그로)

1. **[높음] 시즌 아크 플래너** — 통제군의 승인(勝因)은 사전 설계된 떡밥 대차였다. 온보딩 직후 "약속 발급/회수 시점 표"를 만들고 회차 생성 프롬프트에 현재 위치를 주입하는 장치. 갈림길·진도 인터뷰의 상위 구조.
2. **[높음] 생성 모델 품질이 1차 변수** — provider 선택지(claude 등) 품질 비교 실험.
3. **[중] 폴백 회차의 episode 번호 소모 버그** — 폴백 폐기 후에도 번호가 증가해 표기 드리프트 발생(연속성 -1점·형식 -1.3점의 직접 원인). 결정론 수정 가능.
4. **[중] 후크 패턴 다양성** — 생성 프롬프트에 직전 N화 후크 유형을 주고 "다른 결"을 요구.
5. **[v0.2] StoryScore 개선** — 2글자 이름 변형 의심 위양성(통제군 16건 전부 위양성, "한설"↔"한참") → 최소 길이 가드. 후크 휴리스틱이 문학적 후크를 놓침(양쪽 공통 저점) → 신호 사전 확장.
