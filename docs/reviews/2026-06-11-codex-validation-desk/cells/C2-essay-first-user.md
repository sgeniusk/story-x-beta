## 3. 축별 판정

- 축③ 문체·한국어: 통과. 사물(보호자 침대, 싱크대, 수도꼭지)에서 사유로 넘어가는 에세이 호흡이 자연스럽고, 가족 고발로 단정하지 않는다.
- 축⑤ 기능 UX: 수정. 사실 보호 모드는 유용하게 노출되지만, 에세이 단독 원고 화면에서도 상단 CTA가 `초안 생성`으로 남아 현재 원고 검토/다음 행동이 헷갈린다. 매체 카드 텍스트가 중복되어 자동화 locator도, 초보의 훑어보기에도 일부 모호하다.
