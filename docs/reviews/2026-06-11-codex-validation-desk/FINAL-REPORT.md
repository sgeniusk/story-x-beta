가장 중요한 근거:

- 생성 품질 자체는 기대보다 강하다. C5는 스릴러 완결 지시를 `rewardArc` 4개로 모두 회수했고, C6는 SF 아이디어의 비용을 현재 선택지 축소로 명확히 구현했다.
- C8 CLI 검토는 캐논 충돌을 다수결로 통과시키지 않고 4 blocked + 1 revise로 잡아냈다. 즉 review reasoning은 살아 있지만 editor entrypoint가 막혀 있다.

## 2. 변경사항 제안

P0:

- 없음.

P1 우선 수정:

1. F-002: 출간 잠금 뒤 다음 회차 생성 handler를 복구한다. locked latest chapter를 감지해 다음 episode draft로 넘어가야 한다.
2. F-006: floating editor 작가실 패널의 open state, visibility, z-index, 버튼 렌더 조건을 점검한다. `5명에게 전체 검토 맡기기`는 최소한 command palette나 fallback 버튼으로도 접근 가능해야 한다.
3. F-007: 520px대 좁은 뷰포트에서 핵심 CTA가 `.hx-aside display:none`에 갇히지 않도록 sticky footer 또는 main 하단 CTA를 제공한다.

P2/P3 후속:

- F-001: 인터뷰 대기 시간을 실제 provider 속도에 맞게 안내하고 진행 상태를 노출한다.
- F-003/F-004: 매체 카드 접근성 라벨과 상태별 CTA 라벨을 정리한다.
- F-005: 만화 prompt/quality gate에 선택 포맷과 사용자가 적은 컷 수를 hard constraint로 넣는다.
- F-008: commercial↔literary 축을 온보딩 작품 계약에서 선택하게 하고, editor slider는 project scope 저장임을 명확히 한다.
- F-010: 오디오/학술 draft 직후 one-click 매체별 review smoke를 제공한다.

바꾸지 말아야 할 것:

- 에세이 사실 보호 모드는 유지해야 한다. C2에서 아버지의 직업/나이/병명 등 미작성 디테일을 발명하지 않았다.
- 완결형 draft의 `rewardArc`/`stakesLedger` 기록은 유지해야 한다. C5/C6에서 회수 여부를 판단하는 데 매우 유용했다.
- 오디오 draft의 `[MUSIC]`, `[SFX]`, 쉼, 반복 후렴 표기는 유지해야 한다. C9에서 매체 적합성이 가장 선명하게 보인 부분이다.
- 학술 draft의 `[근거 필요]` 표기는 유지해야 한다. 수치를 발명하지 않는 안전장치로 작동했다.

다음 테스트에서 확인할 지점:

- F-002/F-006/F-007 수정 뒤 C4를 실제 8~12화까지 재실행하고 매 회차 5인 검토, 캐논 누적, 갈림길 카드를 확인한다.
- C8을 UI에서 직접 본문 충돌 주입 → 5인 검토 → memory pending 후보 생성까지 다시 수행한다.
- C7은 별도 새 origin 또는 새 project scope에서 slider를 literary 80~90으로 움직인 뒤 quality gate와 draft prompt가 실제로 달라지는지 확인한다.
- C9는 오디오/학술 draft 이후 매체별 review agent가 실제 본문에 붙는지 확인한다.

## 3. 성장 메모리 업데이트

작품 canon 후보:

- 산장 스릴러: 산장 유리문 안쪽의 작은 손자국은 민호의 마지막 흔적이며 윤재의 거짓 증언을 무너뜨린 핵심 증거다.
- SF 단편: 미래 답장을 읽을수록 현재에서 가능한 선택지가 물리적·상황적으로 줄어드는 비용이 발생한다.
- 오디오 드라마: 713번 막차의 안내 방송에는 다음 날 교체되기 전까지 아이 아빠의 생전 목소리가 남아 있다.

문체/품질 기억:

- 에세이와 문학 단편은 사물의 반복 이미지가 감정 설명을 대신할 때 강해진다. C2의 보호자 침대/싱크대, C7의 컵 바닥 물때가 좋은 표본이다.
- 완결 지시는 `rewardArc`와 `stakesLedger`에 실제 payoff/lost/kept를 기록하게 해야 회수 품질 확인이 쉽다.
- 학술 칼럼은 실제 수치가 없을 때 `[근거 필요]`를 남기고 수치를 발명하지 않는 규칙을 유지한다.

시각/오디오 기억:

- 오디오 드라마는 SFX/MUSIC/침묵/후렴 변주를 본문 표기 레벨에서 유지한다.
- 만화/인스타툰은 선택 포맷과 사용자가 명시한 컷 수를 hard constraint로 저장해야 한다.

실패 로그:

- P1: 출간 잠금 뒤 `초안 생성`이 다음 회차 생성으로 이어지지 않아 장편 루프가 차단된다.
- P1: floating editor 작가실 패널에서 5인 전체 검토 CTA가 나타나지 않아 review 루프가 차단된다.
- P1: 좁은 뷰포트에서 온보딩 CTA가 `.hx-aside display:none`에 갇혀 진행할 수 없다.
- Known issue recurrence: C8 review JSON의 `stderr`에 Codex 배너/경고/스킬 본문이 함께 저장됐다.
