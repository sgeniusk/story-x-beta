# Story X 5인 페르소나 사용자 테스트 — 2026-05-16

Claude Cowork에 5인 작가 페르소나 시뮬레이션을 위임해서 받은 결과 + 우선순위 메모. Vercel team SSO 차단으로 외부 agent가 실제 사이트 접근 불가했고, 인터페이스 설명 기반 *시뮬레이션 모드*로 진행됐다는 점을 감안한다. 카피·라벨·정보 흐름 평가는 유효, 실제 클릭·로딩 체감은 본 테스트에서 측정되지 않음.

## 페르소나 5명 (모두 가상)

- **김지수** (24, 1년차 로판 신인) — 소설 / 장편 — 별 4
- **박영주** (42, 출판 5권 에세이스트) — 에세이 / 회고 — 별 3
- **이수민** (29, 인스타툰 작가) — 만화 / 인스타툰 4컷 — 별 2

평균 3.2/5.0. **분산이 평균보다 더 중요한 신호**. 소설 매체는 4점대, 만화는 2점.

## 공통 막힘 지점

1. **에디터가 모든 매체에 글쓰기 textarea 기본 화면** — 이수민(만화) / 정현우(오디오북). 만화 작가에게는 컷 박스가, 오디오북 PD에게는 플레이 버튼이 첫 화면에 있어야 함.
2. **작가진 5명 카드가 픽션·소설 전제** — 박영주·이수민·정현우 셋 모두 자기 매체의 핵심 검수자가 누구인지 모름.
3. **카피의 소설 편향** — "초안 생성"(에세이/만화/오디오북에 어색), "캐논"(에세이·동화엔 안 씀), "환상로맨스"(김지수: "로판") — 매체별로 라벨이 동적이어야 함.

## 공통 좋아한 지점

1. **자유 서술 → 인터뷰가 내 단어를 받아씀** — 김지수·한승호 강하게 긍정. *"내 작품의 어휘로 도구가 말하는 첫 경험"* (한승호)
2. **포맷 카드의 세분화** — 인스타툰 4컷 / 회고 에세이 / 동요 읽기 / 시리즈 장편 모두 자기 매체가 1급 시민으로 카드에 떠있는 것 자체가 합격점.
3. **dot 색의 즉시성** — 5명 모두 회색·초록·빨강 즉시 이해. *앰버*만 살짝 모호.

## 페르소나 간 의견 충돌

- **회차 lock 의미**
	- 김지수(웹소설): 안전감, 카카오페이지 감각과 일치
	- 한승호(SF): 자산 축적, 시리즈 정본 관리
	- 박영주(에세이): 갇힌 느낌, "출간 확정"이 원고 빼앗는 어조
	- 이수민(만화): 의미 없음, 인스타툰은 발행 후에도 그림 다시 그림
- **자유 서술 단계 효용**
	- 한승호: 결정적, 내 어휘로 도구가 말함
	- 박영주: 양가적, AI가 *적지 않은 디테일*(엄마 직업 "교사")을 발명함 → 회고 에세이 치명적
- **매체 전환 가치**
	- 박영주: 오디오북 미리듣기 좋음
	- 이수민: 만화 *안에서의* 포맷 변형(4컷→세로 스크롤)이 더 답답함
	- 한승호: 80화 뒤 *지연 전환*에 관심

## 다음 개발 우선순위

1. **카피 매체별 동적 치환** — 가장 빠른 effort, 즉시 별점 상승 기대. 4명이 정확 인용. (이번 세션 진행 = Phase 3-quick)
2. **매체별 1급 시민 에디터** — 만화 컷 박스, 오디오북 플레이 버튼, 에세이 사실 보호 모드. *별점 하향이 모두 이 지점*. 별도 큰 Phase.
3. **AI 발명 권한 (논픽션)** — 자유 서술에 없는 디테일은 발명 전 명시적 사용자 승인. 박영주의 *서비스 거부 사유* 수준.

## 핵심 명제("사람의 끌림 + AI의 일관성") 전달 여부

- ✅ 소설 매체(특히 시리즈)에서 전달 완료 — 한승호의 *연속성 미터 + 자동 캐논 추출 + 앰버 펄스가 종족명 충돌 짚어줌* 3가지 결합
- ❌ 에세이·만화·오디오북에서 전달 실패 — 박영주는 *AI 발명*이 일관성과 반대로 작동, 이수민·정현우는 *자기 매체 전담 검수자*가 작가진 카드에 없어서 일관성 약속이 자기 매체에 적용 안 되는 것처럼 느낌
