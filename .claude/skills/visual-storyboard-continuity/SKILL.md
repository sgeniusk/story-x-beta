# Visual Storyboard Continuity

## Principle

Visual stories have canon too: silhouettes, props, locations, page rhythm, carousel rhythm, and panel promises.

## Workflow

1. Read the visual bible: character silhouettes, outfit rules, recurring props, locations, palette, and panel rhythm.
2. Convert each scene beat into panels or scroll segments.
3. Check whether the reader can infer action without extra exposition.
4. Preserve visual anchors across episodes.
5. Write new visual canon facts after approval.

## Output

- panel or cut sequence
- camera distance
- carousel swipe hook when the format is insta-toon
- visual anchors used
- dialogue density warning
- new visual canon facts

## Checks

| Area | Check |
| --- | --- |
| Character | Silhouette, outfit, expression range |
| Space | Location layout, entrance/exit direction |
| Object | Props, weapons, letters, symbolic items |
| Rhythm | Page turn, carousel swipe hook, scroll hook, panel density |
