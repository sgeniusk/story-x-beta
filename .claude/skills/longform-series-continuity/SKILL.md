# Longform Series Continuity

## Principle

Continuity is state. Treat every chapter as a transaction that reads prior canon and writes new canon.

## Workflow

1. Read the story bible: premise, audience promise, characters, world rules, canon facts, open threads.
2. Extract memory anchors that must appear or be respected in the next chapter.
3. Draft only against those anchors.
4. Check contradictions before polishing prose.
5. Write new canon facts after the chapter is approved.

## Required Output

- episode number and title
- memory anchors used
- outline beats
- chapter draft or scene draft
- continuity report
- new canon facts
- changed open threads

## Continuity Checks

| Area | Check |
| --- | --- |
| Character | Desire, wound, voice, relationship state |
| World | Rules, cost, permissions, chronology |
| Plot | Open threads, reveals, episode numbering |
| Genre | Reader promise, pacing, cliffhanger shape |

## Common Mistakes

- Polishing prose before checking contradictions
- Adding world rules because a scene needs convenience
- Making a character confess information their wound would make them hide
- Forgetting to record new facts after a chapter is accepted
